# WAL Pressure Profile

Demonstrates WAL generated by bounded insert, update, and delete phases.

## Run

```bash
make profile-reset PROFILE=wal-pressure PROFILE_SIZE=small
make metrics-sample
```

## What It Shows

- WAL LSN deltas for write phases;
- `pg_stat_wal` counters;
- relation size after logged changes.

The profile intentionally uses ordinary logged tables. It is not a bulk-load
benchmark; use larger sizes only when you want visible WAL movement.
